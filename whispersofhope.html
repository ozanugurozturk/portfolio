Whispers Of Hope

Project Description

First group project of the game programming education in Unity with other disciplines from our school. During this project we used pair programming technique while we were learning how to collaborate with other programmers and support artist & and designers to implement game actors on levels. We For working collaboratively and not to lose progress of others by overwriting our works, we learned the basic principles of GitHub. The most electrifying part of the project was to also creating our own game story at the beginning of the story by mixing up mythological events, our imaginations or the books that we read and inspirations from the games that we have played so far.


Team Size : 14

Game Engine:Unity

Technology: C#

Project Length: 6 Weeks

Personal Contribution

- Implementation of main character's navigation system.

- Creation of variety of enemy actors that guard or patrol in certain area and implementation of AI logic to attack player.

- Implementation of game characters' attributes and their leveling up system.

- Creation and implementation of menu system and GUI on the gameplay.

- Helping the game designers at the end of the project about the 3 different levels' design.
