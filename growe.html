Project Description

This is the 1 of the 3 group unity projects that I have taken part in during my education. We, as a group of students from all the other game related disciplines in Forsbergs Skola, were asked to deliver a grid based mobile platform game. We made brainstorming as a group to decide on the concept with considering the requirements that were stated at the beginning of the project. In the game, players colllects, plant and sell different type of seeds and grow their online plants and place them in grid based map. In the game, seeds and plants require watering and fertilization to survive and grow as a high quality plant to earn higher game point for players, which are listed and displayed in high-score lists, which requires online save system.


Team Size : 17

Game Engine: Unity

Technology: C#

Project Length: 4 Weeks

Personal Contribution

- Creation and implementation of user authentication menu to be able to sign in as guest, sign up for a new account or log in with an already created account to get players in the game with GOOGLE FIREBASE platform.

- Implementation of game stats communication and interaction between online database and the game to be able to save and load the game.

- Implementation of real time flow into game for letting users to play whenever they want to.

- Creating high-score menu and implementing scoring system according to events that happen in the game and items that players have.

- Implementation process of multiple inventory system within game and their GUI.

- Planning and implementing the random rare seed getting system for stabilization of the game.
