Project Description

This is the final project of the education with students from all the diciplines. Every member of the project worked as employees of a small company. We had limited period of time to deliver certain requirements. Teachers took part as owner of the company and senior management, they empower students to handle all the aspects of the project.


Team Size : 40

Game Engine: Unreal 5

Technology: C++ & Blueprint Visual Scripting

Project Length: 10 Weeks

Personal Contribution

- Creation and implementation of base abilities of the main character such as jumping, sprinting or crouching with Game Ability System plug-in.

- Creation and implementation of base attributes of the all characters such as health, stamina, mana with Game Ability System plug-in. Creating interactions between abilities and attributes.

- Creating and implementing leveling up system for players to increase their characters's abilities.

- Implementation of the attribute menu and attribute tree system to allow users to increase their base values such as attack speed, stamina-mana regeneration ratio while their main character gains experience and level.

- During the project, we had sprint review meetings for each 7 groups. I contributed and managed the process for our group presentations at the end of the each week.
